URL Fields

Order and Transaction IDs are required and used to calculate the URL along with the version and merchant ID. In your integration, you would use the values for these fields used in the initiate browser payment request within your code (Process.aspx based on this example) and not expose these to the payer on this page, or pass them as hidden fields.
order.id
Order ID of the existing Initiate Browser Payment transaction you wish to confirm.
transaction.id
Transaction ID of the existing Initiate Browser Payment transaction you wish to confirm.
